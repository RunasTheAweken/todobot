################################################### ################### Todo Bot ############################ ################################################### ################################################v

Todo Bot is a Telegram bot that allows users to manage their to-do list. Users can add, remove, and mark tasks as completed with simple commands.

################################################### ################### Installation ############################## ################################################### #################################

1. Install the required dependencies by running `pip install pyTelegramBotAPI.
2. Clone or download the project files to your local computer.
3. In the `bot.py` file, replace '---YOUR TOKEN---' with your token

################################################### ################### Usage ############################## ################################################### ################################

1. Run the `bot.py` script to start the bot.
2. In Telegram, start a chat with your bot and send the `/start` command to see the available commands.
3. Use the `/add` command to add a new task to your to-do list.
4. Use the `/list` command to view your tasks and their status (completed or not).
5. Use the `/delete` command to remove a task from your to-do list.

################################################### #################### Commands ############################# ################################################### #################################################

- `/start`: Displays a welcome message and shows available commands.
- `/add`: Prompts the user to enter a new task to add to the to-do list.
- `/list`: Displays the user's tasks and their status (completed or not).
- `/delete`: Allows the user to select and delete a task from the to-do list.
