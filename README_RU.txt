##################################################################### Todo Bot ###############################################################################################################################v

Todo Bot - это Telegram-бот, который позволяет пользователям управлять списком дел. Пользователи могут добавлять, удалять и отмечать задачи как выполненные с помощью простых команд.

##################################################################### Установка ################################################################################################################

1. Установите необходимые зависимости, запустив `pip install pyTelegramBotAPI.
2. Склонируйте или загрузите файлы проекта на свой локальный компьютер.
3. В файле `bot.py` замените '---YOUR TOKEN---' на свой токен

##################################################################### Использование ##############################################################################################################

1. Запустите скрипт `bot.py`, чтобы запустить бота.
2. В Telegram начните чат с вашим ботом и отправьте команду `/start`, чтобы увидеть доступные команды.
3. Используйте команду `/add`, чтобы добавить новую задачу в свой список дел.
4. Используйте команду `/list`, чтобы просмотреть свои задачи и их статус (выполнено или нет).
5. Используйте команду `/delete`, чтобы удалить задачу из своего списка дел.

###################################################################### Команды ################################################################################################################################

- `/start`: Отображает приветственное сообщение и показывает доступные команды.
- `/add`: Запрашивает у пользователя ввод новой задачи для добавления в список дел.
- `/list`: Отображает задачи пользователя и их статус (выполнено или нет).
- `/delete`: Позволяет пользователю выбрать и удалить задачу из списка дел.
